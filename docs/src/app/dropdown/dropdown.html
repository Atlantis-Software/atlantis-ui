For and dropdown use : 

      {{dropdownHtml}}

Specification :

We can use button or a to open dropdown

Don't forget to add data-toggle on the button or tag a

Class css :

Class tag Description Info
.dropdown div Define the dropdown Warning : Optional, if we put the content into a div with position : relative
.dropup div Define the dropdown and define to expand to the up The other class don't change.
Warning : Replaces the .dropdown
.dropdown-toggle button, a Define that open dropdown Warning : don't forget to add data-toggle="dropdown", and if we use tag a to add correct data-target
.dropdown-menu ul Format correctly the content of dropdown
.dropdown-menu-right ul Align the menu to de right of its parent
.divider li Create a divider between li
.dropdown-header li Define a header, to create a label sections
.disabled li Desactive a li
