For an collapse use : 

      {{collapseHtml}}

Precision :

You can use button or a to open collapse

Don't forget to add data-toggle="collapse" and use correct href for link and correct data-target for button.

Class css :

Class Tag Description Info
.collapse div Define the collapse Hide per default
.in div Show the collapse per default

For an accordion use : 

      {{accordionHtml}}

Class css :

Class Tag Description Info
.panel-group div Define the accordion Warning : don't forget to add a id
.panel div Define a panel
.panel-heading div Define the header of panel. That open the panel. It's always show
.panel-title h4 or other h Define the title of panel. Warning : don't forget to add correct data-toggle and data-parent to tag a or button, and for the a the correct href
.panel-collapse div Define the content of panel when that's open Warning : add .collapse. Add .in if a panel it's open per default
.panel-body div Define the body of panel, the main content
.panel-default, .panel-primary, .panel-success, .panel-info, .panel-warning, .panel-danger div Define the style of the panel
