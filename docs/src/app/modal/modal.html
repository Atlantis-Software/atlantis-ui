Modal parameters

Parameters Type Description Value Default value
show boolean Define if the modal is open or not true, false false
orientation String Position of the modal left, bottom, right, top left
backdrop string Define if the backdrop is present true, false true.
width value in px or % Define the width of modal Ex : 300px/50% 600px
