Info :

we can add an id to the modal for create a modal link to a border :

  • ModalBottom : create a modal to bottom
  • ModalLeft : create a modal to left
  • ModalRight : create a modal to right
  • ModalTop : create a modal to top

Class css :

Class tag Description
.modal-dialog div Contains all the modal
.modal-header div Define modal header
.modal-title div Define modal title
.modal-body div Define modal body
.modal-footer div Define modal footer
