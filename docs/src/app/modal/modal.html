For an modal use : 

      {{modalHtml}}

Precision :

we can add an id to the modal for create a modal paste to a border :

  • ModalBottom : create a modal to bottom
  • ModalLeft : create a modal to left
  • ModalRight : create a modal to right
  • ModalTop : create a modal to top

Class css :

  • .modal-dialog : contains all the modal
  • .modal-header : define a header where we can put a title or button like close button
  • .modal-title : define the title of the modal
  • .modal-body : define the body of the modal, where we put all information we want
  • .modal-footer : define the footer of the modal, where we can put different button or other things
