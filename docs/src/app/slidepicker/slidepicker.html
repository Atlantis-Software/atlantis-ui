For a slidepicker use : 

      {{slidepickerHtml}}

Class css :

Class Tag Description Info
.slidepicker div Define the slidepicker Warning : Don't forget to add .slidepicker-vertical. The slidepicker-horizontal don't work for the time being
.slidepicker-vertical div Define a vertical slidepicker
.slidepicker-sm div Define a small slidepicker
.slidepicker-lg div define a large slidepicker
.slidepicker-track div Define the track The height of the track is define with the number of option
.slidepicker-handle span Define the handle Warning : add a style="top : number px" with the height of the first options
.slidepicker-label ul Define the option
.active li Active the li.
.slidepicker-input input Define the input that contain the value choice
