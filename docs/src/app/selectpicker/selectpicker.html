Parameters

We need 3 arguments for use selectpicker

-Multiple : Select multiple if true, optional (if not present : false)

-ngModel : Value to bind

a)String if multiple false

b)Array if multiple true

-Value : Option's value, per default take text in selectpicker-option tag
