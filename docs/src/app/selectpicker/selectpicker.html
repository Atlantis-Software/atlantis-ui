For a selectpicker use :

Method 1 : 

      {{selectpicker1Html}}

Method 2 : 

      {{selectpicker2Html}}

Info

The first method is for an static selectpicker

The second method is for an dynamic selectpicker. But need to be activate with javascript like :

$('select').selectpicker();

Class css :

Method 1 :

Class Tag Description Info
.select div Define the selectpicker
.select-toggle button Like dropdown, allow to open selectpicker Warning : don't forget to add data-toggle="select"
.select-text span Display the choice
.select-options ul Define the different options
.divider li Create a divider between option

Method 2 :

Class Tag Description
.selectpicker select Define the selectpicker
.selectpicker-sm select Define a small selectpicker
.selectpicker-lg select Define a large selectpicker
