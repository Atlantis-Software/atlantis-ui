Dropdown :

Settings

Setting Description
title Define the title of the button what open dropdown
Option Allow to send options to dropdown

Options

Attribut Type Description Value Info
orientation String Define the orientation of the dropdown, up or down Up, down Per default : down
alignement String Define the alignement of dropdown right, left Per default : left

Dropdown-option :

Options

Attribut Type Description Value Info
disabled boolean Define if a option is disabled or not true, false Per default : false
type String Define the type of li action, divider, header Per default : action
