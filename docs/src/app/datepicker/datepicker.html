Datepicker

Simple datepicker : 

    this.valueDatePicker = "2017-05-01";

valueDatePicker is the binded value, if you no put value by default it will be empty.
numberOfMonths is the number of month in the calendar, if you no put value by default it will 3.

Example without value

Datepicker range: 

    this.start1 = "2017-05-01";
    this.end1 = "2017-05-22";

start1 et end1 are the binded value, if you no put value by default it will be empty for the 2 dates.
numberOfMonths is the number of month in the calendar, if you no put value by default it will 3.

Example without value
