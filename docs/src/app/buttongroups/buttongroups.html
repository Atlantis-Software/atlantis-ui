Precision :

For add a dropdown, place a .btn-group into another .btn-group with the dropdown.

For different size add class btn-group-lg, btn-group-sm or btn-group-xs

For a vertical button groups use btn-group-vertical instead btn-group

For create a toolbar use a div with class btn-toolbar, and put btn-group into this div.

Warning for use 100% width, use tag a and add btn-group-justified class
