Settings

Attribut Type Description Value Info
indicator boolean Define if we want indicator in bottom of carousel true, false Per default : false
interval number Modify the time between slide 2000 - infinite If 0 or not present the carousel will not slide
pause String | null Define if we want pauses the cycling on mouseenter hover, null Per default : hover
