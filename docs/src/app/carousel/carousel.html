For an carousel use : 

      {{carouselHtml}}

Precision :

Use identic data-target and href than the id of carousel

Don't forget to add data-slide="prev or next" to carousel-control

Class css :

  • .carousel-indicators : use for create indicator pips to bottom of carousel, optional
  • .carousel-inner : Is the information we want to display with carousel like image or text
  • .carousel-caption : Text to explain the slide, optional
  • .carousel-control : Allow to add controls to left and right of the carousel, for change slide
