For an carousel use : 

      {{carouselHtml}}

Precision :

Use identic data-target and href than the id of carousel

Don't forget to add data-slide="prev or next" to carousel-control

Class css :

Class Tag Description Info
.carousel slide div Define the carousel Warning : don't forget to add data-ride="carousel"
.carousel-indicator ol Define the indicator pips to bottom of carousel Optional
Warning : Don't forget to add data-slide-to="number" for all li and to add the default .active
.carousel-inner div The content of carousel
.item div All the content of one slide Warning : don't forget to add .active to the default item. Otherwise, the carousel don't be display
.carousel-caption div Description of the slide Optional
.carousel-control .left/.right a Allow to controls the carousel, go next and previous Warning : don't forget to add data-slide="prev/next"
