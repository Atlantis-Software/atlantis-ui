Class css :

Class Description
.btn Apply base css to button
.btn-default Define standard style
.btn-primary Define primary style, for primary action
.btn-success Define success style, for a validation or successful action
.btn-info Define info style, for a information action
.btn-warning Define warning style, for a caution action
.btn-danger Define danger style, for a dangerous action like erase or exit
.btn-link Define link style, like tag a
.btn-lg Define a large button
.btn-sm Define a small button
.btn-xs Define a extra small button
.btn-block Use 100% width of his parent
.active Display button into active position
disabled Not a class
Disabled a button, add disabled attribute to button
