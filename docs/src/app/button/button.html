Class css :

  • .btn, .btn-default, .btn-primary, .btn-success, .btn-info, .btn-warning, .btn-danger, .btn-link : Change button styles
  • .btn-block : Button take 100 width of the parent element
  • .btn-lg, .btn-md, .btn-sm, .btn-xs : Change button size
  • .disabled : disabled button
