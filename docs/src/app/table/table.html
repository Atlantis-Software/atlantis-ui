For an table use :

Class css :

Class Tag Description
.table table Define the classic style
.table-striped table Add a zebra striping to odd row
.table-bordered table Add borders to all sides of each cells
.table-hover table Add a hover state on rows
.table-condensed table Make tables more compact
  • .table : define to use 100% width and define the border and all padding
  • .table-stripped : use for zebra-stripping, alternate between gray and transparent background
  • .table-bordered : Add borders between all the columns and around the table
  • .table-hover : Add hover effect when mouse are on line
