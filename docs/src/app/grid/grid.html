For a grid use :

We can use different grid for different screen size :

Name Screen size Container width Class css Column width
Extra small devices like phones <768px Auto .col-xs- Auto
Small devices like Tablets ≥768px 750px .col-sm- ~62px
Medium devices like Desktops ≥992px 970px .col-md- ~81px
Large devices like Desktops ≥1200px 1170px .col-lg- ~97px
