Simple datepicker : 

    {{DatepickerSimpleHtml}}
  
    this.valueDatePicker = "2017-05-01";

valueDatePicker is the binded value, if you no put value by default it will be today.
numberOfMonths is the number of month in the calendar, if you no put value by default it will 3.

Datepicker range: 

    {{DatepickerRangeHtml}}
  
    this.start1 = "2017-05-01";
    this.end1 = "2017-05-22";

start1 et end1 are the binded value, if you no put value by default it will be today for the 2 dates.
numberOfMonths is the number of month in the calendar, if you no put value by default it will 3.
