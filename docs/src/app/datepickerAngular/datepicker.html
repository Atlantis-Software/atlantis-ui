Example :

Different size :

For an datepicker range use : 

      {{DatepickerRangeHtml}}

Precision :

We can define different size for the widget. For that just add a class to datepicker tag like datepicker-sm or datepicker-lg

Parameters :

Parameter Type Description
start string (YYYY-MM-DD) start date
end string (YYYY-MM-DD) end date

For an datepicker simple use : 

      {{DatepickerSimpleHtml}}

Parameters :

Parameter Type Description
ngModel string (YYYY-MM-DD) date
