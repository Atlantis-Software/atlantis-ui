Example with plugins

Plugins object

We can use predefined plugins : checkbox and sortable, we just need to use string with plugin name in our array

For use customs plugins, we just need to create a object with the name of the icon, and the function to call on click or other event

Event list

We can send function for different event

  • click
  • dragenter
  • dragover
  • dragleave
  • dragdrop
  • dragstart
  • dragend
  • mousedown
  • mouseenter
  • mouseleave
  • mousemove
  • mouseout
  • mouseover
  • mouseup
  • dblclick

We can define three other function, one for display none the icon with hidden function, one for disable the icon with disabled function and the last for visibility hidden the icon.

Warning: the two first function need to return true or false. The last function need to send visible or hidden;

Different attribute

attribute Type Description Default
label string It's represent the content of a node for shown in tree Nothing
selectable boolean Optional; Allow to deactivate the possibility to select a node true
expanded boolean Optional; Define if the node is expanded or not; false
children Array of nodes Require if we want a sub-tree, all object in children have same syntax that standard node Nothing
disabled boolean Define if a node is disabled or not, that causes to desactive selection and possibility to expand the node false
