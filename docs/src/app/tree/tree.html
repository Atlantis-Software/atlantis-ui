Nodes attribut description :

attribut Type Description
label string It's represent the content of a node for shown in tree
selectable boolean Optional; per default : true; allow to deactivate the possibility to select a node
expandable boolean It's require for a node with node's children
expanded boolean Optional; per default : false; Define if the node is expanded or not;
children Array of nodes Require if we want a sub-tree, all object in children have same syntax that standard node
disabled boolean Define if a node is disabled or not, that causes to desactive selection and possibility to expand the node

An id is automaticly add to add node, this id is define compared to the index in the parent
