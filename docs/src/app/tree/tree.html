Example with plugins

Plugins object

Different plugins type

Name Description
checkbox Basic plugin, add checkbox to node for selection. For disabled on a node we just need to put selectable: false to this node.
sortable Basic plugin, add the possibility to sort the node in the same level. If we want sortable in different level add nestedSortable arguments to the atlui-tree
custom plugin We can define custom plugin with different event callback.

Custom plugins

Attribute Type Description
icon String Define the icon of the plugin, use icon's name in icon docs
click Function(node); node arguments is the node where the event occurs Callback function when the user clicks on the plugin
dblclick Function(node); node arguments is the node where the event occurs Callback function when the user double-clicks on the plugin
mousedown Function(node); node arguments is the node where the event occurs Callback function when the user presses a mouse button over the plugin
mouseenter Function(node); node arguments is the node where the event occurs Callback function when the pointer is moved onto the plugin
mouseleave Function(node); node arguments is the node where the event occurs Callback function when the pointer is moved out of the plugin
mousemove Function(node); node arguments is the node where the event occurs Callback function when the pointer is moving while it is over the plugin
mouseover Function(node); node arguments is the node where the event occurs Callback function when the pointer is moved onto the plugin
mouseout Function(node); node arguments is the node where the event occurs Callback function when a user moves the mouse pointer out the plugin
mouseup Function(node); node arguments is the node where the event occurs Callback function when a user release the mouse button over the plugin
dragenter Function(node); node arguments is the node where the event occurs Callback function when the dragged element enters the plugins
dragleave Function(node); node arguments is the node where the event occurs Callback function when the dragged element leaves the plugins
dragstart Function(node); node arguments is the node where the event occurs Callback function when the user starts to drag the plugin
dragend Function(node); node arguments is the node where the event occurs Callback function when the user has finished dragging the plugin
dragover Function(node); node arguments is the node where the event occurs Callback function when the dragged element is over the plugin
drop Function(node); node arguments is the node where the event occurs Callback function when the dragged element is dropped on the plugin
disabled Function(node); node arguments is the node where the event occurs Callback function for define if the plugin need to be disabled or not.

The function need to return true or false

display Function(node); node arguments is the node where the event occurs Callback function for define if the plugin is display or not.

The function need to return true or false.

hidden Function(node); node arguments is the node where the event occurs Callback function for define if the plugin is hidden or not, the plugin will just had visibility hidden.

The function need to return true or false.
