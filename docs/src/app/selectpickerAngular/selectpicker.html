si on selectionne une valeur dynamiquement par le code il faut que ce soit la meme instance sinon cela ne fonctionne pas. Par exemple pour le bouton array :

Explication code

Selectpicker1 est la valeur bindée. Si pas de valeur par defaut c'est un tableau vide : 

        Selectpicker1 = [];

sinon c'est un tableau de variables (il faut utiliser la même instance).Dans ce cas un , deux, trois et 11111 seront selectionnés par défaut.

        var arrayUnDeuxTrois = ["un", "deux", "trois"];
        var un = 1;
        Selectpicker1 = [arrayUnDeuxTrois, un];

La valeur entre les balises selectpicker-option est du text ou une valeur bindée et l'attribut value est une valeur bindée.

        {{selectpickerOptionHtml}}

Si l'attribut value n'est pas défini il sera égal à la valeur du texte ( dans l'exemple ci-dessous la valeur de la variable AucuneValeur c'est à dire aucune valeur), donc quand pas de valeur pour la value , la valeur est celle du texte. 

        {{selectpickerOptionSansValueHtml}}
