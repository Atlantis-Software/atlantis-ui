Explanation code

Selectpicker1 is the bind value. If not value by default it's a empty array : 

        Selectpicker1 = [];

Otherwise it's a array of variables ( we must use the same instance). In that case on, two, three and 11111 are selected by default.

        var arrayOneTwoThree = ["one", "two", "three"];
        var one = 1;
        Selectpicker1 = [arrayOneTwoThree, one];

The value between selectpicker-option tags is text or a bind value and value attribute is a bind value.

        {{selectpickerOptionHtml}}

If value attribute is not defined , it will be equal to the value of the text ( in the exemple below the value of the variable NoneValue that is to say none value), so when none value for attribute value, the value is that of the text.
