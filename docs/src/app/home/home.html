Getting started :

Step 1 : Install atlantis-ui

npm install atlantis-ui

Step 2 : Import the component Modules, use standard widget and CSS or less

Import the ngModule for each component you want to use with the function forRoot() or in a shared module :

For use standard widget, add with script atlantis-ui-js

For use css, add link to atlantis-ui-css

For use less, add link to less and add less compiler or loader.

In the function forRoot we just need to add a parameters objects for the grid component

Example of parameters :

Parameters :

Parameter Type Description
type string It's the type you want to define, it allow to put default parameters to a type column
alignment string Optional; Change the alignment of the column and all the cell of this column; it's use like css property
pipes Array of pipes Optional; That need to provide first the pipes in your module.
optionsPipe Array of string Should define in same order has pipes; You can have multiple options, that just need to add a array in place of string
transformation function That allow to pass a function that be fire when we modify content into a cell. That allow to have correct value; for incorrect value always return "invalid value"
