For an inputgroup use : 

      {{inputgroupBasicHtml}}

Precision :

We can place addon before or/and after input. For place before, use span before input. For place after, use span after input

For an inputgroup with checkbox or radio use : 

      {{inputgroupCheckHtml}}

Precision :

To use radio remplace type="checkbox" by type="radio"

For an inputgroup with button use : 

      {{inputgroupButtonHtml}}

Precision :

Like the classic inputgroup, we can place button after input

We can place multiple button, we just need to add other button tag

For an inputgroup with dropdown use : 

      {{inputgroupDropdownHtml}}

Precision :

For use dropdown is like a input group with button, but we just add a dropdown after button and add correct data to button
