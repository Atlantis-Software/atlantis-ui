For an navbar use : 

      {{navbarHtml}}

Specification :

We can use a dropdown menu directly in the navbar, like any dropdown

Don't forget to use .navbar class

Class css :

  • .navbar-default : Use the classic color for navbar
  • .navbar-header : Define the navbar header, use for an 'hamburger' menu on small screen and for display a link to the index.
  • .collapse, .navbar-collapse : Use for hide the menu on small screen, to use withe .navbar-header
  • .disabled : disable a link.
