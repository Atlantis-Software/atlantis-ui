All CSS class we can use into navbar

Class for first div or nav tag :

Class tag Description Info
.navbar first div, nav Define the navbar
.navbar-default first div, nav Define the classic color for navbar
.navbar-inverse first div, nav Define inverse color for navbar
.navbar-fixed-top first div, nav Fixed the navbar to top Include a div with .container into navbar.
Warning : Add padding to body, to avoid that navbar overlay other content. (by default, navbar as 50px height)
.navbar-fixed-bottom first div, nav Fixed the navbar to bottom Include a div with .container into navbar.
Warning : Add padding to body, to avoid that navbar overlay other content. (by default, navbar as 50px height)
.navbar-static-top first div, nav Fixed the navbar to top Include a div with .container into navbar.
Warning : Don't need to add padding to body

Class specific to header :

Class tag Description Info
.navbar-header div after nav Define the navbar-header
.navbar-toggle button Use for small screen, to display navbar like a menu Warning : Don't forget to add data-toggle="collapse" and correct data-target
.icon-bar span Show a bar
.navbar-brand a Display the content with more visibility. Usefull for a home link We can use a image with a tag img into a with .navbar-brand class

Class specific to main body

Class tag Description Info
.navbar-collapse div Define the navbar to be collapse if we are in small screen Warning : Don't forget to add .collapse class to the div
.navbar-nav ul Define the link into navbar
.active li Show the link active, also to show the current status
.disabled li, dropdown Desactive a link
.dropdown li Allow to use dropdown into navbar, see dropdown docs
.navbar-form form Allow to use form into navbar.
.navbar-left ul, form, p, button Align the content to left.
.navbar-right ul, form, p, button Align the content to right
.navbar-btn button Vertically center button into navbar
.navbar-text p Allow to use text into navbar
