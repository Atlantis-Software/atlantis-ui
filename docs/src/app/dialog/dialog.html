Dialog

A dialog is a "window" we can drag, resize and focus on. We can have many dialog open at the same time.

In Contrary to the modal, we are not blocked on the dialog. So we can do anything else.

Dialog parameters

Parameters Type Description Value Default value
show boolean Define if the dialog is open or not true, false false
title String Title of the dialog, that's will appear on the top of dialog - none
width number Define the width of dialog - 300
height number Define the height of dialog - 500
isResizable boolean Define if the dialog is resizable or not true, false false
minHeight, maxHeight, minWidth, maxWidth number Define the min and max size of a dialog when the dialog is resize - 100 for min and Infinity for max
container htmlElement Define the container of dialog, for keep dialog into container when we drag him - none
