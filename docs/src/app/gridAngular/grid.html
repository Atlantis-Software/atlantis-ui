Html :

A grid component is like a table so we can use class like table

The ng template atlui-grid-cell-header allow to send custom template for column header

grid parameters:

Parameter Type Description
columns Array of objects Different columns with all default parameters, columns name, and type of the columns
rows Array of objects The value of rows we send to widget
selected Array of objects The value we selected in rows for changes selection in widget
selectedRows Function with $event Allow to collect what we select into widget
onCellChange Function with $event Allow to get modified index an column cell after value change
sort Function with $event Collect all the sort we define in widget, when we click on a column, this update the $event value
headerFixed Boolean Define if we use fixed header table or not
height Boolean Define if we use fixed header table or not

Parameters:

Parameter Type value Description Default
label string any It's the name of the column Nothing
alignment string like the css property Optional; Change the alignment of the column and all the cell of this column Nothing
width String Value + "em" or "px" or "%" Optional; Define the width of the column auto
type string Any you defined in ngx-atlantis-ui module Define default value of the column with the param we send to ngx-atlantis-ui module. string
isSortable boolean true, false Optional; Define if we can sort the column false
isEditable boolean true, false Optional; Define if we can edit the content of the column false
format string dd/MM/yyyy or full for example The date/time components to include, using predefined options or a custom format string. Default is 'shortDate' shortDate
