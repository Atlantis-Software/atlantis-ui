For a modal use : 

      {{modalHtml}}

Modal :

Options

Parameters Type Description Value Info
orientation String Modify the modal to stick to a border left, bottom, right, top Per default : create a standard modal
size String Modify the size of modal large, small Warning : use only with standard modal
fade boolean Define if we use animation or not true, false Per default : false
backdrop string Define if the backdrop is present true, false, static Per default : true. If define to static the modal not close when we click to backdrop

modal-header

Attribut

Attribut Description
title Define the title of the modal, necessary
options Allow to send options to modal.

Options

Parameters Type Description Value Info
closable boolean Define if we can close the modal with a cross true, false Per default : true
